# Assessment Workspace Overview

## Purpose

This folder contains analysis and redesign planning artifacts generated from the first Spring 2026 course iteration.

Primary goal:
- Use first-iteration evidence, delivery outcomes, and feedback to update the course template, especially `SYLLABUS.md`, assignments, lecture sequencing, and supporting resources.

## Working Direction

Current direction for course revision:
- Preserve the historical Spring 2026 evidence in this folder.
- Keep root `SYLLABUS.md` as the active maintained syllabus.
- Integrate validated assessment findings into template-facing course documents.
- Continue cleanup and restructuring of lecture materials, which is still pending.

## Current Priority Sequence

1. Update `SYLLABUS.md` with confirmed assessment outcomes (including SRDMPA/EDOCA framing where appropriate).
2. Reconcile lecture sequence and naming against delivery evidence.
3. Normalize assignment-to-week alignment and reading references for template reuse.
4. Complete resource/lecture-note cleanup and remove obsolete duplication.
5. Introduce earlier game/simulation and strategic reasoning scaffolding as a recurring through-line.
6. Implement modality architecture (residency, synchronous, asynchronous) with explicit rules and return paths.
