## What is Zotero?

Zotero is a tool for collecting and organizing research sources.

You can:

- save articles, books, and websites
- store citation information
- organize sources into collections

In this course, we use Zotero as a **shared research space**.

Access the shared library here:
[SEID 2364 Ethics group library](https://www.zotero.org/groups/6444008/seid_2364_ethics/library)

This library contains:

- course readings
- sources added by other students
- research materials you can use and build on

## Why We Are Using a Shared Library

This course uses Zotero for two related purposes:

- to build a shared pool of research sources across the class
- to help you begin organizing sources for your own project or interests

The group library is collaborative.

That means:

- other students and the instructor can see what you add
- the quality of your citation matters because other people may use it
- clear titles, complete metadata, and useful links help the whole class

---

## Task 1 — Add a Source to the Shared Library

### Step 1 — Find a source

Find one scholarly or credible source related to **harm**.

Examples:

- academic article
- book or book chapter
- credible report
- reputable website

Choose a source that would be useful to another student in the course, not just one that is easy to add.

---

### Step 2 — Add it to Zotero

Add the source to the **shared course library**.

If possible, use Zotero's identifier tools or browser connector so the citation fields are filled in accurately.

---

### Step 3 — Check the citation (Required)

Make sure the entry includes:

- Author
- Title
- Year
- Source (journal, book, website, etc.)

Also include:

- a link to the source OR
- a file attachment (PDF if available)

The citation must be complete and readable.

Before you move on, open the item and check that the title is correct, the author is in the right field, and the item is saved in the shared library rather than only in your personal library.

---

## Task 2 — Create Your Own Collection

### Step 1 — Create a collection

Create a new collection in Zotero.

Name it something related to your project or topic.

Example:

- "AI and Hiring"
- "Social Media Harm"

Use a name that will still make sense later in the semester.

---

### Step 2 — Add your source

Add your selected source to your personal collection.

You may add more sources later.

Your personal collection does not replace the shared library. You should use both:

- the shared library to contribute to the class research space
- your own collection to organize sources around your interests or assignments

---

## What Makes a Good Source Entry?

A good Zotero entry is:

- complete enough that another person can identify and locate the source
- accurate enough that the citation fields are useful later
- organized clearly enough that you can find it again

Avoid adding incomplete or confusing entries if you can fix them first.

---

## Key Idea

You are doing two things:

- contributing to a shared research space
- building your own research path

Both are important.

---

## Common Problems

- Missing author or date

  Edit the item and fill in the missing citation information.

- No link or attachment

  Add a URL or attach the PDF if you have one.

- Source is not credible

  Replace it with a scholarly or clearly reputable source.

- Item saved in the wrong library

  Make sure one copy is in the shared course library and another is in your own collection if needed.

- Collection not created

  Create a personal collection with a clear topic-based name.

---

## What Success Looks Like

- One complete and readable source is added to the shared library
- Your source includes core citation information
- Your own collection is created and clearly named
- Your source is organized and easy to find later
