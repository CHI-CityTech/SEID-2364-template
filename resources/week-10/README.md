# Week 10 Resources

This folder contains the active build materials for Unit 10.

Week 10 is currently being developed as a generative teaching package. Some files are planning documents, some are implementation documents, and some are student-facing support materials.

## Files In This Folder

- [10-01-signal_tracking_game_master_source.md](10-01-signal_tracking_game_master_source.md)
  - Role: master authoring source
  - Status: working design document
  - Purpose: combines the game system and the first pilot content set in one place

- [10-02-wiki_implementation_guide_signal_integrity_activity.md](10-02-wiki_implementation_guide_signal_integrity_activity.md)
  - Role: implementation guide
  - Status: draft deployment document
  - Purpose: explains how the activity should be structured and run in the wiki

- [10-03-signal_card_template.md](10-03-signal_card_template.md)
  - Role: student-facing raw template
  - Status: ready for use
  - Purpose: two clean copy-paste blocks — Part A to create the page, Part B to add each new transition. No instructions embedded. Links to instructions file.

- [10-04-signal_record_instructions.md](10-04-signal_record_instructions.md)
  - Role: student-facing instructions
  - Status: draft
  - Purpose: field-by-field guide for how to fill out the signal record; linked from the template

- [10-05-example_student_signal_cards.md](10-05-example_student_signal_cards.md)
  - Role: worked example
  - Status: instructional support document
  - Purpose: shows what an evolving signal record with stacked transition reports can look like

## Suggested Use

Use the master source as the primary design file.

Use the implementation guide when building or revising the wiki structure.

Use the signal card template and worked example in assignment instructions or as linked support resources for students.

## Notes

- This is a week-specific working folder, not the canonical home for all future reusable course materials.
- If a Week 10 artifact becomes useful across multiple weeks, it can later be promoted to a shared course-level location.
